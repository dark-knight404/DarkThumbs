# DarkThumbs
Thumbnail preview for EPUB, CBZ, CBR, RAR and ZIP image archives

## Built with
Visual Studio 2019 using Windows 10 SDK

## Minimium system requirements
Windows 7/8/8.1/10 (32/64-bit)
